## 长江雨课堂刷课脚本（2026.06更新，通用支持长江雨课堂中的所有课程）

原项目：https://github.com/RHzzz666/yuketang_for_SUCT.git

## **如何使用**
### 1.安装**Tampermonkey**

https://www.tampermonkey.net/

将油猴脚本安装到你的浏览器中（不会可以搜搜教程）

### 2.下载脚本

复制长江雨课堂刷课for_SCUT.user.js

### 3.导入脚本

点击浏览器右上角油猴Tampermonkey图标

<img src="./img/img_1.png" width="500"><img src="./img/img_1.png" width="500">

点击添加新脚本

将复制的脚本复制进去

### 4.成功后效果

成功导入后进入雨课堂对应课程链接，将自动进行跳转播放

再次点击油猴脚本图标会显示脚本正在运作

<img src="./img/img.png">   <img src="./img/img.png">

### 5.进入课程使用

在长江与课堂中，进入对应课程，再**点击刷新**即可使用

### 6.在原脚本上更改了什么

- 适配新版长江雨课堂页面，新增对 `https://changjiang.yuketang.cn/pro/lms/*` 路径的支持。
- 启动后自动切换到“学习内容”页，避免在“学习日志”页误识别章节或考试卡片。
- 重写学习任务识别逻辑，不再依赖旧页面固定 DOM，而是根据 `未开始`、`进行中`、百分比进度、`已完成` 等状态文本识别学习卡片。
- 增加可见元素过滤，避免扫描隐藏 DOM 导致误点旧页面残留内容。
- 增加点击跳转锁和点击冷却，防止在学习内容页反复点击同一个任务卡片导致页面卡住或触发异常状态。
- 优化视频播放逻辑，自动静音、设置倍速，并在播放过程中定时保活。
- 优化视频完成判断，结合页面完成度、视频播放进度、视频结束状态，以及接近完成后回到开头的情况判断任务完成。
- 完成视频后自动返回学习内容页并继续处理下一项任务。
- 将原来的阻塞式 `alert("已完成！")` 改为非阻塞 Toast 提示，避免弹窗阻塞脚本继续运行。
- 清理冗余状态变量和重复的视频状态重置逻辑，使脚本结构更清晰。
